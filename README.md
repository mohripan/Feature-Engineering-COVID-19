# Feature-Engineering-COVID-19

## Opening
A coronavirus is a group of viruses that can cause disease in animals or humans. Several types of coronavirus are known to cause respiratory infections in humans ranging from cold coughs to more serious ones such as Middle East Respiratory Syndrome and Severe Acute Respiratory Syndrome. A new type of coronavirus has been found to cause COVID-19.
